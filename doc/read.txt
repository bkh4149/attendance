概要
　生徒の出席、下校時間の管理を行います
  登校時に体温を測定し、体温も入力できるようになっています
  途中で電源が切れても復活できます
    すでに入力された分のデータはファイルに残っているので復活できます
　毎日のデータをcsv形式で蓄積していきます
    操作は999→1
　まだ、生徒が学校に残っている場合は赤い文字で表示されます
  毎日の登下校時刻が変化するような場合も対処可能です
    操作は999→2　登校時刻の変更
    操作は999→3　下校時刻の変更

前提
  Pythonのほかpygameのインストールが必要です
  pip install pygame  

起動方法
  ファイルを解答後、そのフォルダに入って
  python t.py

生徒番号を入力します。
　とりあえずデモ版では10人くらいをサンプルデータとして登録してあります
　なので0-9に数字を入れてください

999をおすと先生用のモードに切り替わります
  １：csvデータ出力
  ２：登校時刻の変更
  ３：下校時刻の変更
  ８：999を抜ける
  ９：終了

文字が化けるとき  
　日本語フォントが対応していない可能性があります、その場合はまずpythonの対話モードを起動します
   print(pygame.font.get_fonts()) で使えるフォントを確認します
      たとえば、[,,,hg明朝bhgp明朝bhgs明朝b,,,,]と表示されたら
   プログラム中のfont=の部分を以下のように書き換えます
　　　font = pygame.font.SysFont("hg明朝bhgp明朝bhgs明朝b", 15)

excelの文字が化けるとき
  Pythonの出力はutf8がデフォルトなのですが、excelはsjisがデフォルトになっています
    そこでcsvファイルをsjisに変換してあげればokです
